Project Overview

The project is a solver for the one dimensional cut problem (subproblem of Cutting stock problem http://en.wikipedia.org/wiki/Cutting_stock_problem).
It uses the simplex and the knapsack algorithm for the solution.

The compilation instructions can be found in "Istruzioni Compilazione.rtf".

The project is commented in italian
